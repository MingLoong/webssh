# WebSSH

## 项目简介

WebSSH 是一个基于Go(后端)和Vue2(前端)的Web端SSH连接工具，集成SFTP文件管理

## 功能介绍

- **Web 终端**：通过浏览器直接连接远程服务器，支持一键生成快捷链接。
- **多种认证方式**：支持密码和密钥两种 SSH 登录方式。
- **文件管理**：支持远程文件的上传、下载与浏览。
- **多标签页**：可同时管理多个 SSH 连接会话。
- **初始命令**：支持登录后自动执行指定命令。
- **安全认证**：可选开启 Web 端登录认证。
- **SFTP文件管理**：支持SFTP文件管理，支持复制粘贴等操作。
